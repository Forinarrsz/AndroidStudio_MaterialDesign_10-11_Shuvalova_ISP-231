Приложение на Android, демонстрирующее применение принципов **Material Design 3** с использованием Jetpack Compose. Проект выполнен в рамках учебной лабораторной работы.

## Описание проекта
Приложение отображает анимированный список студентов. Каждая карточка содержит:
- Аватар студента в круглой форме.
- Имя и возраст с использованием кастомной типографики.
- Интерактивную кнопку развертывания для просмотра детального описания с плавной анимацией.

## Реализованные функции (Material Design)

### 1. Цветовая схема (Color Scheme)
- Настроены **светлая** и **тёмная** темы на основе зелёного базового цвета (#006C4C).
- Использованы роли цветов: `Primary`, `Surface`, `Secondary` и др.

### 2. Типографика (Typography)
- Интегрированы кастомные шрифты из Google Fonts:
  - **Montserrat**: используется для имен (Bold) и основного текста (Regular).
  - **Abril Fatface**: используется для крупных заголовков.

### 3. Формы (Shapes)
- **Small (50.dp)**: применено для создания идеально круглых аватарок.
- **Medium**: кастомная форма карточек со скруглением 16.dp только в диагональных углах (topEnd и bottomStart).

### 4. Анимация
- Реализована плавная анимация изменения размера карточки при развертывании (`animateContentSize`) с использованием пружинного эффекта (`spring`).

## Технологии
- **Язык**: Kotlin
- **UI Framework**: Jetpack Compose
- **Design System**: Material Design 3
